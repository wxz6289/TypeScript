# 04 · 函数与对象（Functions & Object Types）

> 对应官方：[More on Functions](https://www.typescriptlang.org/docs/handbook/2/functions.html)、[Object Types](https://www.typescriptlang.org/docs/handbook/2/objects.html)

---

## 1. 函数类型注解

```ts
type Add = (a: number, b: number) => number;

function add(a: number, b: number): number {
  return a + b;
}
```

---

## 2. 可选参数与默认参数

```ts
function f(x: number, y?: number, z = 0) {
  return x + (y ?? 0) + z;
}
```

---

## 3. 函数重载

**实现签名**必须兼容**所有声明签名**：

```ts
function createDate(timestamp: number): Date;
function createDate(y: number, m: number, d: number): Date;
function createDate(a: number, b?: number, c?: number): Date {
  if (b === undefined || c === undefined) {
    return new Date(a);
  }
  return new Date(a, b, c);
}
```

---

## 4. `void` 与返回值

```ts
function noop(): void {
  console.log('side effect');
}
```

---

## 5. 对象类型：索引签名

```ts
type Dict = {
  [key: string]: number;
  length: number; // 若存在具体属性，须与索引签名类型兼容
};
```

**注意**：`number` 索引的返回值须是 `string` 索引返回值的子类型（JS 会将数字索引转为字符串）。

---

## 6. 泛型函数（与 05 衔接）

```ts
function first<T>(arr: T[]): T | undefined {
  return arr[0];
}
const n = first([1, 2, 3]); // number | undefined
```
