# 第 3 章自测

**Q1** 为什么 `const o = { x: 1, y: 2 }` 推断 `x,y` 为 `number` 而非字面量？  
**思路**：widening；需要 `as const` 或 `satisfies`。

**Q2** 解释「上下文类型」在回调参数推断中的作用。  
**思路**：`map` 回调参数由外层泛型约束。

**Q3** 写判别联合表示「加载中 / 成功 / 失败」三态，并说明非法组合如何被排除。  
**思路**：`{s:'loading'}|{s:'ok',data}|{s:'err',msg}`。

**Q4** `async`/`await` 相比层层回调对类型流的好处？  
**思路**：`Promise<T>` 链式返回类型可追踪。

**Q5** 「新推断站点」直觉题：何时用类/柯里化帮助推断？  
**思路**：当中间类型参数无法从单一表达式推断时，拆步提供锚点。
