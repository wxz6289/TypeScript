# 第 6 章自测

**Q1** 何时用 `readonly` 修饰元组比 `ReadonlyArray` 更合适？  
**思路**：固定长度 + 位置语义（如 `[x,y]`）。

**Q2** `const` 断言与 `readonly` 在「字面量收窄」上的分工？  
**思路**：`as const` 深冻结字面量；`readonly` 常配合接口形状。

**Q3** 用 `Record` 建模动态键对象时，如何避免「任意字符串键」过宽？  
**思路**：`keyof` 自有限集合；模板字面量键；`satisfies`。

**Q4** 解释「映射类型 + 条件类型」生成「所有键可选除 id」的一种写法思路。  
**思路**：`Omit` + `Partial` 或 `Exclude<keyof T,'id'>` 映射。

**Q5** 何时 `enum` 仍可能被团队接受？  
**思路**：与遗留协议数字对齐、需反向映射；否则倾向联合字面量。
