# 08 · 声明文件与 JavaScript 互操作

> 对应官方：[Declaration Files](https://www.typescriptlang.org/docs/handbook/declaration-files/introduction.html)、[Type Checking JavaScript Files](https://www.typescriptlang.org/docs/handbook/type-checking-javascript-files.html)

---

## 1. 为什么需要 `.d.ts`

- 为**已有 JS 库**补充类型信息，使 TS 项目获得智能提示与检查。
- 发布 TS 库时常同时发布 `.d.ts`（`declaration: true`）。

---

## 2. 环境声明（Ambient）

```ts
// global.d.ts（示例）
declare const __BUILD_VERSION__: string;
```

---

## 3. 在 `.js` 中做类型检查：`checkJs`

`tsconfig` 中：

```json
{
  "compilerOptions": {
    "allowJs": true,
    "checkJs": true
  }
}
```

---

## 4. JSDoc 类型注解（节选）

```js
/**
 * @param {string} name
 * @returns {number}
 */
function len(name) {
  return name.length;
}
```

官方 JSDoc 参考：[JSDoc Reference](https://www.typescriptlang.org/docs/handbook/jsdoc-supported-types.html)。
