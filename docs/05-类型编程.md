# 05 · 类型编程（Type Manipulation）

> 对应官方：[Creating Types from Types](https://www.typescriptlang.org/docs/handbook/2/types-from-types.html) 系列

---

## 1. 泛型（Generics）

**在定义处保留类型参数，在使用处传入具体类型**：

```ts
interface Box<T> {
  value: T;
}
const numBox: Box<number> = { value: 1 };
```

约束：`T extends U`

```ts
function longest<T extends { length: number }>(a: T, b: T): T {
  return a.length >= b.length ? a : b;
}
```

---

## 2. `keyof` 与索引访问类型

```ts
type Point = { x: number; y: number };
type K = keyof Point; // "x" | "y"

type P = Point['x']; // number
```

---

## 3. `typeof` 类型查询

```ts
const user = { name: 'Ada', age: 36 };
type User = typeof user;
```

常用于从常量/函数推断类型，减少重复书写。

---

## 4. 条件类型（Conditional Types）

```ts
type IsString<T> = T extends string ? true : false;
type A = IsString<'hi'>; // true
type B = IsString<1>; // false
```

**分布式条件类型**（distributive）：`T extends ...` 中若 `T` 为联合，会分发到每个成员。

---

## 5. 映射类型（Mapped Types）

```ts
type ReadonlyPoint = { readonly [K in keyof Point]: Point[K] };
```

---

## 6. 模板字面量类型（Template Literal Types）

```ts
type Lang = 'en' | 'zh';
type Route = `/${Lang}/home`; // "/en/home" | "/zh/home"
```

---

## 7. 内置工具类型（Utility Types）

官方 Reference：[Utility Types](https://www.typescriptlang.org/docs/handbook/utility-types.html)

常用速记：

| 工具 | 作用 |
|------|------|
| `Partial<T>` | 全部属性可选 |
| `Required<T>` | 全部属性必选 |
| `Readonly<T>` | 全部只读 |
| `Pick<T, K>` | 选取键 |
| `Omit<T, K>` | 排除键 |
| `Record<K, T>` | 键到值的映射对象 |
| `Exclude<T, U>` | 从联合 T 去掉可赋给 U 的成员 |
| `Extract<T, U>` | 从联合 T 取出可赋给 U 的成员 |
| `NonNullable<T>` | 去掉 null / undefined |

```ts
type Todo = { title: string; desc?: string; done: boolean };
type Patch = Partial<Pick<Todo, 'title' | 'done'>>;
```
