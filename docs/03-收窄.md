# 03 · 收窄（Narrowing）

> 对应官方：[Narrowing](https://www.typescriptlang.org/docs/handbook/2/narrowing.html)

---

## 1. 什么是收窄

从**宽类型**（如 `string | number`）通过控制流分析，缩小到**窄类型**（如 `string`），从而安全调用该类型上的成员。

```ts
function padLeft(s: string, padding: string | number) {
  if (typeof padding === 'number') {
    return ' '.repeat(padding) + s;
  }
  return padding + s;
}
```

---

## 2. 常用收窄手段

| 手段 | 示例 |
|------|------|
| `typeof` | `typeof x === 'string'` |
| `truthiness` | `if (x)` / `if (x != null)` |
| `equality` | `x === 'a' \|\| x === 'b'` |
| `instanceof` | `err instanceof Error` |
| `in` | `'kind' in shape` |
| 自定义类型谓词 | `function isFish(pet: Pet): pet is Fish` |

```ts
type Fish = { swim: () => void };
type Bird = { fly: () => void };

function isFish(pet: Fish | Bird): pet is Fish {
  return (pet as Fish).swim !== undefined;
}
```

---

## 3. 判别联合（Discriminated Union）

**可判别字段**（常为 `kind` / `type`）+ `switch` 是实践中最稳的模式之一：

```ts
type Circle = { kind: 'circle'; radius: number };
type Square = { kind: 'square'; side: number };
type Shape = Circle | Square;

function area(s: Shape) {
  switch (s.kind) {
    case 'circle':
      return Math.PI * s.radius ** 2;
    case 'square':
      return s.side ** 2;
  }
}
```

---

## 4. `never` 与穷尽性

若 `switch` 未覆盖所有联合成员，未覆盖分支中 `s` 可能为 `never` 以外的类型；使用 `assertNever` 可在编译期强制补全分支（见 [01-基础](./01-基础.md)）。
